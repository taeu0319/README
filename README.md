# 🌐 개인 포트폴리오 웹사이트 (Personal Portfolio Website)

동국대학교 WISE캠퍼스 컴퓨터공학과 김태우의 개인 포트폴리오 및 자기소개 웹페이지입니다. 
웹 개발의 기본인 HTML과 CSS를 활용하여 직관적이고 깔끔한 자기소개 페이지를 구축했습니다.

## 📂 파일 구성 (File Description)

이 리포지터리는 웹페이지를 구성하는 핵심 파일들로 이루어져 있습니다.

### 1. `index.html` (메인 웹페이지)
웹사이트의 뼈대와 콘텐츠를 담고 있는 메인 HTML 파일입니다.
* **프로필 및 자기소개:** 개발자로서의 목표와 현재 학적 사항(동국대학교 WISE캠퍼스 컴퓨터공학과)을 소개합니다.
* **기술 스택 (Skills):** 현재 학습 중이거나 활용할 수 있는 프로그래밍 언어(C/C++, Python, Java 등)와 웹 기술(HTML/CSS)을 나열합니다.
* **프로젝트 (Projects):** 학업 중 진행한 주요 과제 및 개인 프로젝트의 아카이빙 공간입니다.
* **연락처 (Contact):** 이메일 및 깃허브 등 외부와 소통할 수 있는 링크를 제공합니다.

## 🛠️ 기술 스택 (Tech Stack)
* **Markup:** HTML5
* **Style:** CSS3 (CSS Grid, Flexbox, Custom Properties)

## 🚀 실행 방법 (How to Run)
1. 이 저장소를 로컬 환경에 클론(Clone)하거나 다운로드합니다.
2. 다운로드한 폴더에서 `index.html` 파일을 더블 클릭하여 웹 브라우저(Chrome, Edge 등)로 엽니다.
